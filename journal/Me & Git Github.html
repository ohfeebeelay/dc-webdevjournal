Now that you’ve learned a bit about Git and Github, reflect upon and answer the following questions.

What is the difference between Git and Github?
Git is installed on your local machine and it helps you track and maintain files in the Git repository
Github is a cloud based service that helps you host and manage the repositories
Why use Git?
Flexible
Why use Github?
Collaborative
What is the payoff to using Git and Github?*
You get the best of both worlds.
